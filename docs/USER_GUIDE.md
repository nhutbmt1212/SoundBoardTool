## ğŸ†˜ Troubleshooting

### App doesn't start
**Solution:**
- Right-click SoundboardPro.exe
- Select "Run as Administrator"
- Check antivirus (may flag as false positive)

### VB-Cable won't install
**Solution:**
- Run SoundboardPro.exe as Administrator
- Or download manually: https://vb-audio.com/Cable/
- Install and restart computer

### Discord doesn't hear sounds
**Solution:**
1. Check Discord Input Device = "CABLE Output"
2. Check Input Volume is not muted
3. Ensure "â–¶ï¸ Start Routing" is active in soundboard
4. Try leaving and rejoining voice channel

### Sounds don't play
**Solution:**
- Check volume slider is not at 0
- Check sound file is valid
- Try different sound file

### "CABLE Input" not in device list
**Solution:**
- Restart computer after VB-Cable install
- Restart SoundboardPro.exe
- Check VB-Cable is installed in Control Panel

## ğŸ’¡ Tips & Tricks

### Organize Sounds
- Name files clearly (e.g., "airhorn.wav", "laugh.mp3")
- Keep sounds in one folder for easy management

### Volume Control
- Adjust volume before joining voice chat
- Don't blast people's ears!
- Test with friends first

### Sound Quality
- Use .wav for best quality
- .mp3 works great for most sounds
- Keep files under 10MB for quick loading

### Discord Setup
- Test in a private channel first
- Adjust Input Sensitivity if needed
- Use Push-to-Talk for better control
